# 🏪 متجر الريان | Al-Rayyan Store

متجر إلكتروني احترافي ومتجاوب بالكامل (Responsive E-Commerce Website) تم تطويره باستخدام **HTML, CSS, JavaScript** مع تصميم فاخر مستوحى من المتاجر العالمية، يركز على تجربة المستخدم وسهولة التسوق مع دعم اللغة العربية واتجاه RTL.

---

## ✨ نظرة عامة | Overview

متجر الريان هو واجهة متجر إلكتروني حديثة تعرض مجموعة متنوعة من المنتجات تشمل:

* الأجهزة الكهربائية
* الإلكترونيات
* الأثاث المنزلي
* الأدوات المنزلية
* السلع الاستهلاكية

يوفر الموقع تجربة تسوق سلسة مع سلة مشتريات متكاملة، نظام مفضلة، أكواد خصم، طلب مباشر عبر واتساب، وتأثيرات تفاعلية حديثة.

---

## 🚀 المميزات الرئيسية

### 🛍️ إدارة المنتجات

* عرض المنتجات بشكل احترافي
* تصنيف المنتجات حسب الأقسام
* البحث الفوري داخل المنتجات
* فرز المنتجات حسب:

  * السعر تصاعدي
  * السعر تنازلي
  * نسبة الخصم
  * الاسم

### ❤️ المفضلة (Wishlist)

* إضافة المنتجات للمفضلة
* حفظ البيانات داخل LocalStorage
* عرض المنتجات المحفوظة لاحقًا

### 🛒 سلة التسوق

* إضافة المنتجات للسلة
* تعديل الكميات
* حذف المنتجات
* تحديث الأسعار تلقائيًا
* حفظ السلة محليًا

### 🎟️ نظام أكواد الخصم

الأكواد المدعومة:

| الكود     | الخصم           |
| --------- | --------------- |
| WELCOME10 | خصم 10%         |
| RAYYAN20  | خصم 20%         |
| SAVE50    | خصم ثابت 50 ر.س |
